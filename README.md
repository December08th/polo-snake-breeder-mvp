# PyThrone MVP - Snake Breeder Management

A feedback collection page for the PyThrone snake breeder management system MVP.

## Overview

This static site allows Polo to review Phase 1 features and submit feedback. Form submissions are handled by Netlify Forms.

## Features Covered

### Phase 1 (Active - Collecting Feedback)
- Snake Profiles
- Collection View
- Feeding Log
- Weight Log
- Pairing Manager
- Clutch Tracker
- Basic Genetics

### Phase 2 (Preview)
- Genetics Calculator
- Family Tree / Pedigree
- Shed Tracking
- Public Breeder Profile
- Waitlist System
- Multi-language Support

### Phase 3 (Preview)
- Marketplace Listings
- Payment Processing
- Shipping Integration
- Buyer Messaging

## Deployment

This site is deployed on Netlify. Form submissions are available in the Netlify dashboard.

## Local Development

Simply open `index.html` in a browser to preview the site.
